# ADR-003 - Governanca de Delivery

## Contexto
Projeto novo precisa de rastreabilidade com baixo overhead.

## Problema
Escolher ferramenta de planejamento e execucao integrada ao desenvolvimento.

## Opcoes
- GitHub Projects
- Jira
- Notion

## Decisao
GitHub Projects como fonte oficial de backlog e status.

## Consequencias
- Positivas: conexao nativa com issues/PRs/releases.
- Negativas: menos recursos de PM enterprise.
- Mitigacao: padronizar campos e ritos de cadencia.
