# Sprint 01 (M1 Setup)

## Objetivo da Sprint
Entregar fundações tecnicas do produto com monorepo funcional, baseline de auth, design system e pipeline.

## Entregáveis
- Estrutura `apps/admin`, `apps/pwa`, `apps/api`, `packages/core`, `packages/ui`
- Endpoints iniciais do contrato API
- Documentação de produto, arquitetura e governança
- Workflow de CI e template de PR

## Definition of Done da Sprint
- `pnpm lint`, `pnpm test`, `pnpm build` passando localmente
- Documentos principais versionados
- Backlog importavel pronto
